# DiscordBot

Basic Discord bot, this one can do:

  1- Says hello.

  2- Sends info about the server.
  
  3- Sends cat images.
  
  4- Sends Pokémon and allows users to register them in their Pokédex.
  
  5- Users can see how many Pokémon they have collected so far.

Till now, more updates are comming.
